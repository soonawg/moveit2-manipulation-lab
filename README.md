# MoveIt2 Manipulation Lab

## 개요
MoveIt2 Manipulation Lab은 로봇 조작 및 환경 상호작용을 위한 실험 및 시뮬레이션을 수행하기 위한 플랫폼입니다. 이 프로젝트는 모듈식으로 설계되어 다양한 로봇 작업을 지원합니다.

## 기능
- 로봇 팔 조작
- 장애물 회피 및 경로 계획
- 상호작용 가능한 환경 시뮬레이션
- 다양한 로봇 모델 지원

## 프로젝트 구조
- `src/`: 소스 코드
- `launch/`: 로봇 시뮬레이션 및 실행 파일
- `config/`: 구성 파일 및 설정
- `scripts/`: 유틸리티 스크립트

## 노드 설명
- `manipulation_node`: 로봇 팔의 동작을 제어하는 노드입니다.
- `planning_node`: 경로 계획 및 장애물 회피를 처리하는 노드입니다.
- `simulation_node`: 로봇과 환경을 시뮬레이션하는 노드입니다.
- `interface_node`: 사용자 인터페이스와 상호작용하는 노드입니다.

이 프로젝트는 MoveIt2 프레임워크를 기반으로 하며, 다양한 로봇 시나리오를 테스트할 수 있는 유연성을 제공합니다.
